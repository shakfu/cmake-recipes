# About

Portable sample app using CMake and CPack.

The app uses two dynamic libraries, one of them using yet another library:

```
PortableApp
  libLibOne.dylib
  libLibTwo.dylib
    libLibThree.dylib
```

Tags: CMake, CPack, MacOS X bundle, .dmg, portable app.  

# Building

```
mkdir build
cd build
cmake ..
cpack -G DragNDrop
```
